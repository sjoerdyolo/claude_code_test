# Personal AI Web App

A modern web application featuring an onboarding flow and personalized AI workspace.

## Features

### Onboarding Flow
- 3-step personalized onboarding
- Collects personal and company information
- Beautiful gradient UI with progress tracking

### Main Application

#### AI Agent
- Real-time chat interface with AI assistant
- Interactive canvas for brainstorming and note-taking
- Split-screen layout for optimal productivity

#### Personal Data
- View and edit personal information
- Manage company details
- Secure local storage of preferences

#### Top Knowledge Workers
- Marketplace of industry thought leaders
- Follow/unfollow functionality
- Filter by followed users
- Expert profiles with bios and expertise tags

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- CSS3 with modern gradients and animations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
